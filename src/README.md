# Amyloid Modifier for ChimeraX

**Amyloid Modifier** is a okay-ish powerful, interactive ChimeraX tool designed specifically for structural biologists and cryo-EM researchers working with multi-layer amyloid fibrils and complex filamentous polymers. 

Processing amyloid models often involves tedious manual labor: fixing broken chain IDs, trimming ragged fibril ends to create clean publishable models, and setting up complex restraints for molecular dynamics or ISOLDE refinement. Amyloid Modifier automates this entire workflow in a unified, user-friendly GUI.

## Key Features

* **Intelligent Layer Detection:** Automatically detects protofilaments and horizontal layers using customizable spatial parameters (Z-Shift and XY-Shift limits), supporting both parallel and anti-parallel amyloid arrangements.
* **Automated Trimming:** Easily trim your fibril to a specific number of central layers. The tool intelligently retains associated waters and ions while discarding ragged edges.
* **Smart Chain Renaming:** Systematically rename chains from the middle out (e.g., A, B, C... or 0, 1, 2...) to ensure standardized nomenclature. Automatically resolves duplicate or broken chain IDs.
* **ISOLDE Restraint Generation:** Instantly export a `.cxc` command file to apply ISOLDE torsion restraints. It automatically restrains the outer layers of the fibril using the central layer as a structural template.
* **Secondary Structure Re-evaluation:** Seamlessly triggers ChimeraX's built-in `dssp` algorithm to recalculate and annotate exact $\beta$-sheet and $\alpha$-helix boundaries after trimming.
* **PDB & mmCIF Support:** PDB is typically for trimming and renaming the model, CIF is for trimming the best layers from a broken model generated by ModelAngelo.

## Installation

**Via ChimeraX Toolshed (Recommended):**
1. Open ChimeraX.
2. Go to `Tools` -> `More Tools...`
3. Search for **Amyloid Modifier** and click **Install**.

**Manual Installation:**
1. Download the latest `.whl` release.
2. In the ChimeraX command line, run:
   `toolshed install /path/to/ChimeraX_AmyloidModifier-1.0-py3-none-any.whl`

## Quick Start Guide

1. **Launch the Tool:** Go to `Tools` -> `Structure Editing` -> `Amyloid Modifier`.
2. **Load a Model:** Open a model in ChimeraX, then click **Load and Copy from ChimeraX** in the tool. This safely creates a working copy so your original file is untouched.
3. **Adjust Parameters (if necessary):** * *Z Shift Min/Max:* Adjust the vertical distance expected between stacked subunits (default: 4.6Å - 5.0Å).
   * *XY Shift Limit:* Adjust the maximum horizontal twist/drift allowed between stacked chains.
   * *Anti-parallel Mode:* Check the "Anti" box if your fibril consists of anti-parallel pairs.
4. **Trim Layers:** Enter the number of central layers you wish to keep and click **Trim Layers**.
5. **Rename Chains:** Click **Rename Chains** to standardize your nomenclature from the center outwards.
6. **Evaluate:** Click **Re-evaluate $\beta$-sheet structure** to update the secondary structure headers.
7. **Export Restraints:** (Optional) Click **Generate Restrain File** to export an ISOLDE `.cxc` script.
